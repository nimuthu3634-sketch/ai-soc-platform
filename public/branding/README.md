# Runtime Branding Assets

Place the primary Aegis Core branding assets here for the frontend shell:

- `public/branding/aegiscore-logo.svg`
- `public/branding/aegiscore-mark.svg`
- `public/branding/aegiscore-mark.png`
- `public/branding/aegiscore-logo.png` (optional raster export)

The React application treats the SVG files as the canonical transparent assets.
The PNG files are optional raster exports if you need them for external tools
or slide decks.
